# Tiger & Hunters

A two-player asymmetric pursuit game played in your browser.

## How to Play

**Goal:**  
- **Tiger:** Incapacitate all 5 Hunters  
- **Hunters:** Form a triangle around the Tiger in the Clearing

**Controls:**  
- Click a piece, then click where you want it to move  
- Tiger moves first each round  
- Hunters move sequentially in any order

## Setup for GitHub Pages

1. Create a new repository on GitHub
2. Upload all files/folders
3. Go to Settings &gt; Pages
4. Source: Deploy from a branch (main branch, /root folder)
5. Save and wait ~1 minute
6. Your game will be live at `https://[your-username].github.io/[repo-name]/`
